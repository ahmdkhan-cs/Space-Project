# Space project


### First of all

- cd server/src
- cp config_example.js config.js
- inside the config.js file add your MongoDB database URL



### To run the project

- Goto root directory
- On windows (npm install then npm run install)
- On mac / linux (npm run install)
- npm run watch


### To make a production build 

- Goto root directory
- npm run deploy
- cd server
- npm start


### To run test cases

- Goto root directory
- npm run test


#### Project Developed Using

- JavaScript
- Node.js
- React.js
- MongoDB
